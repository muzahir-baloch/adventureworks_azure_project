# AdventureWorks Azure Project

Learning focused Azure data engineering project using the AdventureWorks database.

Goals:
- Dev, Test, Prod environments using low cost Azure resources
- CI/CD with Azure DevOps (multi stage)
- GitHub as source of truth for SQL, pipelines, and infra as code (later with Bicep)

Folder structure:
- infra: infrastructure as code templates (Bicep)
- sql: database schema and seed scripts
- datafactory: Data Factory pipeline exports
- pipelines: Azure DevOps YAML definitions
- notebooks: any supporting notebooks or scripts
- docs: architecture diagrams and notes
